### What Ive Built

####  Sketch-to-Level Generator
AI & computer vision pipeline that converts **hand-drawn maps** into fully playable **Unity levels** using **U-Net segmentation**, synthetic data generation, and procedural instantiation.

**Tech:** Computer Vision 路 U-Net 路 Image Segmentation 路 Unity 路 Procedural Generation

---

####  Movie Recommender System (Web App)
End-to-end movie recommender implementing **collaborative filtering and matrix factorization**, evaluated using RMSE, coverage, and diversity metrics.  
Deployed as an **interactive Next.js web application**.

**Tech:** Python 路 Machine Learning 路 Recommender Systems 路 Next.js 路 Data Pipelines

---

####  Edgar Allan Poe Text Generator (GPT-2)
Fine-tuned a **GPT-2 language model** on Edgar Allan Poes literary works to generate stylistically consistent prose.  
Includes training, evaluation using perplexity and qualitative analysis, and a live **Hugging Face demo**.

**Tech:** PyTorch 路 Transformers 路 NLP 路 Text Generation 路 Model Evaluation

---

####  Astronomical Object Classifier
Supervised machine learning system classifying **STAR / GALAXY / QSO** objects from SDSS photometric data, achieving **~97% accuracy** using ensemble models.

**Tech:** Scikit-Learn 路 Feature Engineering 路 Classification 路 Data Mining

---

####  Tweet Moderation System
NLP pipeline for detecting harmful or offensive content using preprocessing, vectorization, perplexity analysis, and downstream classification models.

**Tech:** NLP 路 Text Classification 路 Data Cleaning 路 Model Evaluation
